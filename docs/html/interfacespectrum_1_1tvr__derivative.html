Detailed Description

Computes an estimate to the derivative of an evenly-sampled data set using total variation regularization.

Syntax
real(real64)(:) function tvr_derivative( &
real(real64) dt, &
real(real64) x(:), &
real(real64) alpha, &
optional integer(int32) maxiter, &
optional real(real64) tol, &
optional integer(int32) niter, &
optional class(errors) err &
)
Parameters
[in]dtThe time step between data points.
[in]xAn N-element array containing the data whose derivative is to be estimated.
[in]alphaThe regularization parameter.
[in]maxiterThe maximum number of iterations to allow. The default is 20 iterations.
[in]tolThe convergence tolerance to use. The tolerance is applied to the difference in Euclidean norms of the derivative update vector. Once the norm of the update vector is changing less than this tolerance, the iteration process will terminate. The default is 1e-3.
[out]niterThe number of iterations actually performed.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • SPCTRM_MEMORY_ERROR: Occurs if a memory allocation error occurs.
  • SPCTRM_SINGULAR_MATRIX_ERROR: Occurs if the internal Hessian estimate becomes singular.
Returns
An N-element array containing the estimate of the derivative.
